صمد المکچی — مهندس ارشد دواپس و معمار نرم‌افزار با بیش از ۲۰ سال تجربه در طراحی، توسعه و استقرار سیستم‌های تحت وب.

تماس با من مشاهده مهارت‌ها

درباره من

سلام 👋 من صمد المکچی هستم.
مهندس نرم‌افزار با بیش از ۲۰ سال تجربه در توسعه وب، با تخصص اصلی در بک‌اند، DevOps و توسعه سیستم‌های مدرن مبتنی بر میکروسرویس. در طول سال‌ها، نقش‌های کلیدی در پروژه‌های مختلف به عنوان مدیر DevOps، اسکرام مستر و توسعه‌دهنده ارشد ایفا کرده‌ام و همواره در تلاش بوده‌ام تا فرآیندهای توسعه، تست و استقرار نرم‌افزار را با استفاده از Docker، Docker Compose، Kubernetes، CI/CD و ابزارهای اتوماسیون بهینه و خودکار کنم.
علاقه‌مند به طراحی و پیاده‌سازی معماری میکروسرویس‌ها، توسعه APIهای مقیاس‌پذیر، و سیستم‌های پیچیده با قابلیت انعطاف بالا هستم. در زمینه توسعه فرانت‌اند و تجربه کاربری، با React، Next.js و سایر فریمورک‌های مدرن جاوااسکریپت کار کرده‌ام و توانایی ایجاد برنامه‌های وب مقیاس‌پذیر و سریع را دارم.
در زمینه تست و کیفیت نرم‌افزار، از BDD، TDD و DDD بهره می‌برم و با استفاده از پایتون، Jest و ابزارهای خودکارسازی تست، تست‌های جامع و قابل اعتماد ایجاد می‌کنم. متدولوژی‌های چابک و اسکرام برای من بخش حیاتی توسعه نرم‌افزار هستند و به عنوان اسکرام مستر، هماهنگی و هم‌افزایی تیم را کلید موفقیت پروژه‌ها می‌دانم.
به مستندسازی و به اشتراک‌گذاری دانش اهمیت ویژه‌ای می‌دهم و تجربه کار با ابزارهایی مانند Swagger برای API، PHPDoc برای PHP و JSDoc برای JavaScript را دارم تا کدها و پروژه‌ها به شکل حرفه‌ای و قابل فهم برای تیم و جامعه توسعه‌دهندگان مستندسازی شوند.
از نظر شخصی، فردی آرام، پرانرژی و خلاق هستم که تعاملات مؤثر تیمی را فرصت‌هایی برای یادگیری و رشد می‌دانم. در حال حاضر، دانش خود را در Kubernetes و ارکستراسیون کانتینرها گسترش می‌دهم تا توانایی مدیریت سیستم‌های پیچیده و توزیع‌شده را افزایش دهم.

مهارت‌ها و تخصص‌ها

🔧 تخصص در DevOps و زیرساخت

تجربه گسترده در مدیریت زیرساخت، کانتینرسازی، اتوماسیون CI/CD، و مانیتورینگ سیستم‌ها.
در حال یادگیری Kubernetes و Vault برای مدیریت ارکستراسیون کانتینرها و اسرار امن، و همچنین Terraform و AWS برای زیرساخت به عنوان کد و ابری.

🛠️ مهارت 📝 توضیحات
Docker / Docker Compose کانتینرسازی و محیط‌های ایزوله برای توسعه و استقرار برنامه‌ها
Ansible اتوماسیون پیکربندی و مدیریت زیرساخت
GitHub Actions / CI/CD اجرای pipelineهای خودکار برای build, test و deploy
Prometheus & Grafana جمع‌آوری متریک‌ها و ساخت داشبوردهای مانیتورینگ
Loki & Tempo لاگ‌گذاری مرکزی و Distributed Tracing برای میکروسرویس‌ها
Traefik معکوس‌پراکسی، لود بالانسر و مدیریت روتینگ درخواست‌ها
Kubernetes (در حال یادگیری) ارکستراسیون و مقیاس‌پذیری کانتینرها
Vault (در حال یادگیری) مدیریت امن اسرار و کلیدها
Terraform (در حال یادگیری) Infrastructure as Code برای مدیریت زیرساخت‌ها
AWS (در حال یادگیری) مدیریت زیرساخت ابری و خدمات مرتبط
Selenium / Pytest تست‌های خودکار و تضمین کیفیت نرم‌افزار

💻 توسعه نرم‌افزار، معماری بک‌اند و ابزارها

تخصص در توسعه بک‌اند مقیاس‌پذیر با استفاده از میکروسرویس‌ها و معماری پاک، طراحی API مدرن (RESTful و GraphQL)، و تضمین کیفیت با BDD، TDD و DDD. استفاده از فریم‌ورک‌ها و کتابخانه‌های پیشرفته برای توسعه سریع، تست، مستندسازی و مصورسازی داده‌ها در پروژه‌های Python, JavaScript و PHP.
در حال یادگیری بهترین الگوهای طراحی، ابزارهای جدید و کتابخانه‌های پیشرفته برای بهبود کارایی و کیفیت کدنویسی.

PHP Laravel, CodeIgniter, Symfony, Composer, PHPUnit, PHPDoc, Eloquent ORM, طراحی و توسعه سرویس‌ها
JavaScript / Node.js Express.js, Socket.io, RESTful & GraphQL API, Next.js, React.js, Jest, Mocha, مستندسازی با JSDoc

Design Patterns & Architecture Clean Architecture, CQRS, Event Sourcing, Repository Pattern
Testing & Quality Unit, Integration, Functional, E2E Tests، تضمین کیفیت سرویس‌ها و APIها

API Tools Postman, Swagger، تست و مستندسازی API

🧠 مدیریت پروژه چابک و مهارت‌های نرم

تمرکز بر رهبری تیم، مدیریت پروژه‌های چابک، و توسعه مهارت‌های فردی و بین‌فردی برای افزایش بهره‌وری و هماهنگی تیم.
